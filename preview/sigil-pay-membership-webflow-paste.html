สมัครเข้าสู่ SĪGIL Access

SĪGIL คือชั้น private access ของ MMD สำหรับสมาชิกที่ต้องการเข้าถึง model, ข้อมูล, และระบบดูแลแบบเป็นส่วนตัวมากขึ้นครับ

หน้านี้ใช้สำหรับสมัครและชำระค่า access เท่านั้น หลังชำระแล้วทีมจะตรวจสอบและเปิดสิทธิ์ตามแพ็กเกจที่เลือก

VIP / Black Card จะไม่เปิดสมัครตรงจากหน้านี้ ต้องผ่าน private review หรือ invitation เท่านั้นครับ

SĪGIL Rule Notes
  • Membership payment เป็นการชำระเต็มจำนวนเท่านั้น
  • Standard / Premium มีอายุ 365 วัน ส่วน 7-Day Entry มีอายุ 7 วัน
  • 100 บาท = 1 point ตามระบบ points ledger
  • VIP / Black Card ต้องผ่าน owner review และไม่ใช่การซื้ออัตโนมัติ

2) Private Code / Discount Code

โค้ดส่วนลดควรอยู่ใน payment gate นี้ ไม่ใช่หน้าเลือกแพ็กเกจ
ถ้ามี campaign code หรือ private code ให้กรอกที่นี่ ระบบจะตรวจสอบก่อนคำนวณยอด

3) วิธีการชำระเงิน

ระบบจะส่ง payment method ไปยังหน้า confirm เพื่อแนบหลักฐานต่อ
หน้านี้ไม่รับ deposit สำหรับ membership ครับ ระบบจะส่งรายการเป็น payment_stage = membership

4) ข้อมูลสำหรับเปิดสิทธิ์

ใช้เท่าที่จำเป็นสำหรับ official verification และเปิด access
ช่วยรักษาความเป็นส่วนตัวและใช้เรียกในระบบ
ระบบจะสร้างรายการ membership intent และแนบ code / package / amount ไปให้ทีมตรวจสอบต่อ
SĪGIL CHECKPOINT
